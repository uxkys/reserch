Shape Puzzle Game 使い方ガイド

1. アプリ概要

ランダムに配置された4種類の図形（●▲■★）のうち、正解の図形を素早く当てるゲームです。

「GoodUI / BadUI」を切り替えながら、どちらが操作しやすいか比較することも可能です。

2. ゲーム開始

  1. 起動時に表示されるスタート画面で、プレイ時間（秒）を入力します。
  2. 「Start」ボタンをクリックすると、ゲーム画面が表示されます。
  3. タイマーがスタートし、制限時間内にできるだけ多くのレベルをクリアしてください。

3. GoodUI モードの遊び方

  • 画面下部のグレー帯に4つの図形ボタンが表示されます。
  • 正解だと思う図形ボタンをクリックしてください。
  • 正解なら右側に「⭕️」が3秒表示され、次のレベルに進みます。
  • 不正解なら「❌」が3秒表示されます。

4. BadUI モードの遊び方

  1. ハンバーガーメニューから「UI切り替えボタン」を押してBadUIに変更します。
  2. 下部のグレー帯は消え、「Answer」ボタンが表示されます。
  3. 「Answer」を押すと4種類の図形が並ぶモーダルが出るので、選択＆Yesで回答確定です。
  4. 不正解でもポップアップは表示されません。（廃止済）

5. 終了とCSV出力

  • 制限時間切れ、または全レベルクリア（200レベル）が終わるとゲーム終了です。
  • 「Session Ended」のメッセージが表示され、画面右下に「Download CSV」ボタンが出現します。
  • クリックすると、各レベルのクリアタイムやUIモードなどが記録されたCSVをダウンロードできます。

6. Restart / 強制終了

途中でやり直す場合は、画面左下にある「Restart」ボタンを押すとセッションを強制終了し、再スタートできます。
